# GEODOS
A DOS GEOMETRY DASH PORT WITH EDITOR WRITTEN IN C.

COMPILE:

GRAB THE COMPILER
https://github.com/andrewwutw/build-djgpp/releases

UNARCHIVE IT

COMPILE IT!
djgpp/bin/i586-pc-msdosdjgpp-gcc -O2 -o GEODOS.exe GEODOS.c -lm

THEN SAME SHI FOR THE EDITOR
djgpp/bin/i586-pc-msdosdjgpp-gcc -O2 -o EDITOR.exe EDITOR.c -lm

HAVE FUN!!!

TESTED ON THINKPAD 600 WITH 256MB OF RAM
